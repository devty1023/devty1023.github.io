diving into hakyll

Posted on May 31, 2014 by devty

This blog is proudly powered with hakyll, an open source static site generator written in haskell.

Among a plethora of open source static site generators, hakyll caught my attention for two reasons.

One, haskyll is the only open source static site generator written in haskell with an active user and developer community. In fact, I think it is arguably one of the more active project among all open source static site generator project.

Two, haskyll has huge potential for growth, i.e. it is not a featured packed as other static site generators[1][2] but simple enough for extensive cutomization.

This is all great since I’ve recently become intrigued with functional programming and had set my on learning and using haskell as my primary language. Having an open source project I want to contribute is a huge booster for me to commit to a language, and hakyll fits this bill perfectly.

Some of the features I will be looking to implement in hakyll:

  • Native theme support (easy way to browse and install new themes)
  • Permalinks
  • dynamic index page where laste N post are shown
  • “draft” mode
  • clean url /slug
  • My own theme
